Daji Agent Studio

一个基于 FastAPI、Tool Calling、Skills、Memory 与 Todo 的本地 AI Agent 工作台。

查看项目演示 PDF 查看 GitHub 仓库

项目功能

  • 支持网页端多轮对话，提供本地 AI Agent 工作台界面。
  • 支持 Tool Calling，可调用文件查看、文件读取、安全命令执行等工具。
  • 支持 Skills 技能包、长期记忆 Memory、Todo 任务规划与聊天历史持久化。
  • 使用 FastAPI 提供后端接口，原生 HTML/CSS/JavaScript 实现前端页面。

技术栈

Python FastAPI JavaScript HTML/CSS OpenAI-compatible API Tool Calling Git / GitHub

本地运行方式

克隆项目后，配置 .env 文件并安装依赖，运行： 

uvicorn backend.main:app --reload

启动后访问：http://127.0.0.1:8000/app
